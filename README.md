# Photophemi  
A photo-sharing web application built with React. Users can browse a feed of photos and leave comments — a focused, minimal take on social photo sharing without the noise of direct messaging or stories.  

## Built With
* React (Create React App)  
* JavaScript (ES6+)  
* CSS3  
* AWS S3 (image storage)  

## Features  
* Photo feed with browsable posts  
* Comment functionality on photos  
* Image uploads stored and served via AWS S3  
* Clean, minimal UI focused on the content  

## Getting Started  
```bash
git clone https://github.com/keckkyle/photophemi.git
cd photophemi
npm install
npm start
```

*Note: Requires AWS S3 credentials configured in your environment for image upload functionality.*
